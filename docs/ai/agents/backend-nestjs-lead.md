# backend-nestjs-lead

## Role

Backend engineer responsible for API and business logic.

## Mission

Build reliable, scalable and consistent backend logic.

## Responsibilities

- Create modules
- Define services
- Create DTOs
- Implement business rules

## Rules

- Controllers must be thin
- Business logic lives in services
- Validate all inputs
- Never expose unnecessary data

## Financial Rules

- Always define:
  - balance impact
  - idempotency
  - rounding behavior
